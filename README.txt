REST backend jonne voi tallentaa tietoja käyttöliittymällä.

REST backend toteutettu MongoDB:llä ja Node.js:llä
Käyttöliittymä toteutettu Reactilla

LAAJENNUKSIA
Express: Luo palvelimen  
Nodmon: Uudelleen käynnistää palvelimen, kun koodia muutetaan. 
Mongoose: Ohjailee MongoDB:tä

Käyttö:
Potkaise MongoDB ylös (windowssilla [mongon asennus kansio]/[versio]/bin/mongod.exe)
Potkaise Node.js palvelin ylös ()
Potkaise React-UI ylös.
